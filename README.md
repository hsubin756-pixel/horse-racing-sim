# 경마 시뮬레이션 / 競馬シミュレーション

경마를 주제로 Python을 공부하며 만든 프로젝트입니다.  
競馬をテーマにPythonを学びながら作ったプロジェクトです。

## 기능 / 機能
- 경주 시뮬레이션 (스태미나, 피로, 전술 시스템) / レースシミュレーション（スタミナ・疲労・戦術システム）
- 혈통 기반 능력치 보정 / 血統による能力値補正
- 라이벌 등록 시스템 / ライバル登録システム
- FastAPI 기반 REST API / FastAPIベースのREST API

## 사용 기술 / 使用技術
- Python
- FastAPI
- SQLite

## 실행 방법 / 実行方法
```
uvicorn main:app --reload
```
